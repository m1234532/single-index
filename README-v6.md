# 单身狗基因测序 v6.0

## 🧬 作品介绍

一个基于Vue 3的单页面互动人格测试应用，采用"基因检测"概念包装，通过28道题目分析用户的"单身基因"，生成专属的基因证书报告。

**核心差异化**：36种人格类型 × 36套专属配色方案，真正的千人千面视觉体验。

---

## ✨ 功能特性

- 🧬 **基因检测主题**：DNA双螺旋动画、实验室风格、证书式报告
- 🎨 **36套配色方案**：每种人格有独特的视觉主题
- 📊 **SVG雷达图**：四维人格分析可视化
- 📸 **分享卡生成**：一键生成精美分享图片
- 🎭 **28道测试题**：涵盖社交、择偶、心动、独处四个维度
- 💫 **流畅动效**：页面切换、按钮交互、浮动动画
- 📱 **移动端优先**：完美适配手机端体验

---

## 🎨 视觉系统

### 设计原则

采用"人格色彩映射"系统，四种维度决定配色：

| 维度 | 影响 | 示例 |
|------|------|------|
| 社交 | 主色调温度 | 活跃=暖色，保守=冷色 |
| 择偶 | 饱和度 | 完美=高饱和，开放=低饱和 |
| 心动 | 明度 | 敏感=明亮，冷静=沉稳 |
| 独处 | 辅助色 | 独立=对比色，依赖=邻近色 |

### 配色示例

- **极光猎人 (APSI)**：深蓝紫渐变 + 霓虹青
- **温室花朵 (APSD)**：粉白渐变 + 薄荷绿
- **快乐小狗 (AOSD)**：暖黄 + 珊瑚红
- **古堡隐士 (RPSI)**：墨黑 + 紫水晶

## 🎯 使用流程

1. **闪屏页**：观看DNA动画加载，点击"进入实验室"
2. **欢迎页**：了解测试说明，点击"开始基因测序"
3. **答题页**：回答28道题目（每题4个选项）
4. **结果页**：查看专属基因证书，包括：
   - 人格类型码（如APSI）
   - 人格名称和emoji
   - 单身指数
   - 四维雷达图
   - 详细解析和建议
5. **分享**：点击"生成分享卡"保存图片

## 🔧 自定义配置

### 添加新题目
编辑 `js/data.js` 中的 `QUESTION_BANK` 数组：

```javascript
{
  id: 29,
  text: "新题目文本",
  emoji: "🎯",
  dim: "social", // social/filter/heart/alone
  options: ["选项A", "选项B", "选项C", "选项D"]
}
```

### 添加新配色
编辑 `js/visual-system.js` 中的 `personalityThemes`：

```javascript
'NEWCODE': {
  name: '新人格名称',
  colors: {
    primary: ['#主色1', '#主色2'],
    secondary: ['#辅色1', '#辅色2'],
    bg: '背景渐变',
    // ...
  },
  emoji: '🎭'
}
```
