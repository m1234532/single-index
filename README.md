# 你的单身指数

基于专业爱情心理学框架的单身诊断H5应用。

## 项目结构

```
single-index/
├── index.html          # 主入口文件
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── data.js         # 题目和结果数据
│   └── app.js          # Vue应用逻辑
├── assets/             # 静态资源（如有）
└── README.md           # 项目说明
```

## 技术栈

- Vue 3 (CDN)
- Vant 4 (CDN)
- html2canvas (CDN)
- 原生CSS3动画

## 功能特性

- 加载进度条动画
- 16道心理学测试题
- 4维度分析（社交/滤镜/心动/独处）
- 81种结果类型
- Emoji风格头像
- 结果海报生成
- 深色/浅色模式切换
- 礼花动画效果

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 部署

上传所有文件到任意静态托管服务（GitHub Pages / Cloudflare Pages / Vercel 等）。
