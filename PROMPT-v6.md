# Prompts赛道参赛作品：单身狗基因测序 v6.0

## 作品概述

**作品名称**：单身狗基因测序 (Single Dog Gene Lab)  
**核心创意**：将MBTI人格测试与基因检测概念结合，通过28道题目分析用户的"单身基因"，生成专属的人格报告和基因证书。  
**设计亮点**：36种人格类型 × 36套专属配色方案，真正的"千人千面"视觉体验。

```markdown

你是一位精通Vue.js、CSS动画和视觉设计的全栈前端工程师，也是一位深谙色彩心理学的设计专家。

创建一个名为"单身狗基因测序"的互动人格测试网页，具备以下特征：

# 视觉设计系统

## 色彩理论
采用"人格色彩映射"系统：
- 社交维度决定主色调温度（活跃=暖色，保守=冷色）
- 择偶维度决定饱和度（完美=高饱和，开放=低饱和）
- 心动维度决定明度（敏感=高明度，冷静=低明度）
- 独处维度决定辅助色（独立=对比色，依赖=邻近色）

## 36种人格配色示例

3. 自由灵魂(AOSI): 深紫 + 荧光粉 (#2D1B4E → #F093FB)
4. 快乐小狗(AOSD): 暖黄 + 珊瑚红 (#FFFBE6 → #FF6B6B)
5. 月光骑士(VPSI): 深海蓝 + 月光银 (#0F2027 → #64FFDA)

## 设计思路

### 为什么选择"基因检测"主题？
1. **话题性**：基因检测是近年热点，容易引起好奇
2. **科学性**：给人"专业测试"的心理暗示
3. **可视化**：DNA双螺旋、实验室元素天然适合视觉设计
4. **分享欲**："我的基因报告"比"我的测试结果"更有传播力

### 为什么36种人格要36套配色？
- 市面上大多数测试只用1-2套配色
- 我们希望每个用户都有"专属感"
- 色彩心理学：不同人格适合不同色彩
- 技术挑战：证明动态主题系统的可行性
